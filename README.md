# 🚀 An LLM-Powered SQL Automation Assistant!

This project focuses on building an interactive AI chatbot that understands natural language and automates SQL tasks, making database operations **faster, simpler, and more intuitive**.

Using **LangChain**, **LangGraph**, **MCP servers for tool calling**, and **Streamlit for the frontend**, I built a smooth chat-based interface where users can work with their database just by describing what they want.

---

## 💡 How It Works

- The user connects to their database by providing credentials  
- Once connected, they can simply type instructions in **plain English**  
- The assistant understands the request and generates the appropriate **SQL query**  
- It can execute queries, fetch data, and simplify repetitive tasks  
- No manual query writing required — **unless you want to!**

---

## 🔧 Tech Stack Highlights

- **LLM Model:** Groq’s `gpt-oss-120b`  
- **Frameworks:** LangChain + LangGraph  
- **Tool Calling:** MCP servers  
- **Frontend:** Streamlit  
- **Current Database Support:** MySQL  
- **Coming Soon:** SQL Server & PostgreSQL

---

## 🎯 Project Goal

The goal is simple — **make database operations faster, smarter, and accessible to everyone.**

By combining **LLMs with structured tool calling**, this assistant allows users to interact with databases through **natural language instead of writing complex SQL queries manually**.
